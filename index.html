SAMMCharts - OpenSAMM Javascript Vizualization Library

SAMMCharts

SAMMCharts is a Javascript visualization library for the Software Assurance Maturity Model (SAMM) version 1.0.

GitHub: https://github.com/qudosoft-labs/SAMMCharts

Examples:

  • Scorecard as standalone Javascript
  • Roadmap with RequireJS
  • Roadmap with Current Phase Overlay
